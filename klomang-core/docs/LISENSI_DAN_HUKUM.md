# Lisensi dan Aspek Hukum Klomang Core

## Ringkasan Lisensi

Klomang Core didistribusikan di bawah Lisensi Apache 2.0. Lisensi ini memberikan hak kepada pengguna untuk menggunakan, memodifikasi, dan mendistribusikan kode sumber perangkat lunak ini secara bebas, dengan syarat-syarat tertentu yang tercantum dalam file LICENSE di direktori root.

## Batasan Tanggung Jawab (Disclaimer)

Perangkat lunak Klomang Core disediakan "sebagaimana adanya" (as-is), tanpa jaminan apa pun, baik secara eksplisit maupun implisit, termasuk namun tidak terbatas pada jaminan kelayakan untuk diperdagangkan, kesesuaian untuk tujuan tertentu, atau tidak adanya pelanggaran hak cipta. Tim pengembang Klomang Core tidak bertanggung jawab atas segala kerugian finansial, kehilangan data, atau konsekuensi hukum yang timbul dari penggunaan perangkat lunak ini.

## Status Testnet

Klomang Core saat ini berstatus "Testnet Ready", yang berarti perangkat lunak ini telah diuji secara ekstensif dalam lingkungan testnet. Penggunaan Klomang Core pada jaringan utama (mainnet) sangat tidak disarankan tanpa audit keamanan tambahan oleh pihak independen. Risiko penggunaan di mainnet sepenuhnya ditanggung oleh pengguna.

## Pemberitahuan Hak Cipta

Hak cipta atas Klomang Core dipegang oleh Klomang Core Developers, tahun 2026. Semua hak dilindungi undang-undang.
