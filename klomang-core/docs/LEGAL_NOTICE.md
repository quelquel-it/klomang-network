# Pemberitahuan Hukum

## Pengantar
Klomang Core adalah implementasi mesin BlockDAG yang siap produksi, dirancang untuk penerapan skala besar. Dokumen ini menyediakan informasi hukum penting terkait penggunaan perangkat lunak ini.

## Implementasi Teknologi
Kode sumber Klomang Core mencakup implementasi algoritma konsensus GHOSTDAG untuk pengurutan blok paralel dalam struktur BlockDAG, serta pohon keadaan Verkle Trees untuk manajemen keadaan yang efisien dengan kompleksitas operasi O(log n).

## Mekanisme Keamanan Virtual Machine
Virtual Machine berbasis WebAssembly (WASM) yang diintegrasikan dalam Klomang Core dilengkapi dengan mekanisme keamanan sandbox yang memastikan isolasi eksekusi kontrak pintar, mencegah akses tidak sah ke sumber daya sistem, dan menjaga integritas jaringan.

## Penafian Risiko
**Penting:** Status Klomang Core saat ini adalah "Testnet Ready", yang berarti perangkat lunak ini telah diuji secara ekstensif dalam lingkungan testnet namun belum dioperasikan di jaringan utama (mainnet). Penggunaan Klomang Core di jaringan utama merupakan risiko yang ditanggung sepenuhnya oleh pengguna. Pengembang tidak bertanggung jawab atas kerugian finansial, kehilangan data, atau konsekuensi hukum yang timbul dari penggunaan perangkat lunak ini di lingkungan produksi.
